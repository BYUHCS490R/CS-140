CSS

Explain the difference between inline, embedded, and external CSS.

Inline css is when you add css in the html tag. Embedded css is when you put it in the head tag. External css is when you have your css code in a different file and it's linked in the head tag.

Explain why you should only use external CSS.

Allows the html page to not have heavy content for the page to load faster. It also allows you to use the css file for different pages.

What is a selector used for?

A selector targets a certain html element to modify.
