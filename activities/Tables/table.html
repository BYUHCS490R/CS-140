Tables

When do you use a table?

When we have table to display. Could be schedule or tasks or scoreboards.

When do you not use a table?

When the information is not tabular. For example, we don't use a table for layout or for lists of links.

When do you need to do extra work for table accessibility?

We need to label the rows when the cells are merged.

Why don't we use the border attribute?

Because it's obsolete in html so we should use css to style it instead.

What is the easiest way to color every other row a different color?

Use nth-child in css
