CSS Grid

1. Please describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.

It refers to the hierarchy where one element (the parent) contains other elements (the children). This relationship is essential in CSS Flexbox and Grid because the layout behavior of child elements is influenced by the properties applied to the parent container.

2. What are the two main differences between CSS Grid and CSS Flex?

Grid is 2D and as a parent, when using grid, it determines the size of the child element. Flex tries to give the child how much space it can get.
