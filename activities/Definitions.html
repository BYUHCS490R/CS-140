Internet and Web Definitions

Internet

The internet is the global network of computers that are connected to each other.

Web

The web is a service that runs on the internet and contains webpages.

How are the Internet and Web different?

The internet connects computers while the web is found on the internet.

ASCII and Unicode

ASCII is a 7-bit character set containing 128 characters.Unicode is a universal system that gives every character, symbol, or emoji a unique number so computers can display text from any language.

What is UTF-8

UTF-8 is a way to encode Unicode characters using one to four bytes.

URL

A URL is the address of a specific webpage or file on the internet.

Domain Name

A domain name is a human-readable address that points to an IP address of a server where a website is hosted.

HTTP

HTTP is a protocol used for transferring data over the web.

HTTPS

HTTPS is a secure version of HTTP that encrypts data for safe transmission.

IP Address

An IP address is a unique string of numbers separated by periods that identifies each computer using the Internet Protocol to communicate over a network.

DNS

DNS is a system that translates human-friendly domain names into IP addresses that computers use to identify each other on the network.

W3C Recommendations

W3C Recommendations are standards and guidelines developed by the World Wide Web Consortium to ensure the long-term growth of the web.

XML, XHTML, HTML, HTML5

XML is a markup language that defines rules for encoding documents in a format that is both human-readable and machine-readable. XHTML is a stricter version of HTML that follows XML syntax rules. HTML is the standard markup language used to create web pages. HTML5 is the latest version of HTML that includes new features and improvements for modern web development.
