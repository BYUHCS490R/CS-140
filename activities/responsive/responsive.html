Responsive Pages

Why is the viewport meta tag important?

It helps the website look good on all screen sizes.

How can you adjust the CSS for printing?

Create a new css file with all the css and link it as a print media.

When would you want to adjust the CSS for printing?

When the page is supposed to be printed such as tickets, coupons or QR codes.

Why do we need a viewport meta tag?

To control the layout on mobile browsers.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

"flex-direction:row;" takes priority because it's the last command to be executed.

How do you change the CSS for a web page displayed on a phone?

Use @media and modify the html based on the screen size.
