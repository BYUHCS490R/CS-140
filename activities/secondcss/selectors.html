CSS Selectors

1. List out the different ways to select an item.

By selecting the tag name, class name, ID name or with parent/child tags.

2. What is the difference between an ID and a Class?

When we use ID we cannot use it more than once while for class you are allowed to use the same class name for multiple elements.

3. What is the difference between a SPAN tag and a DIV tag?

A span tag is an inline tag while a DIV tag is a block tag and causes line breaks.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A div tag is a generic container while header, main, and footer define specific parts of a webpage.

5. What are the different ways to select a color?

You can select a color by using color names, hex codes, rgb values or hsl values.
